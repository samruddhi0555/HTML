# Laundry Mart Website

## Overview

Laundry Mart is a simple and user-friendly laundry service website built using HTML. The website provides information about laundry services, pricing, and allows customers to submit their details through a booking form.

## Features

* Welcome page with introduction to Laundry Mart
* Service information section
* Laundry service image display
* Price list table
* Customer booking form
* Contact information in the footer

## Technologies Used

* HTML5

## Project Structure

```
Laundry-Mart/

```

## Services Offered

* Dry Cleaning
* Washing and Cleaning
* Ironing and Folding
* Stain Removal
* Express Service

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
4. The Laundry Mart website will be displayed.
